# 🧠 AI-Based Animal Behavior Analyzer

## 🚀 Overview
An AI-powered tool that analyzes animal images/videos to detect species and predict behavior.

## 🎯 Features
- Species Detection
- Behavior Classification
- Alert Level Scoring
- Research Insight Generation

## 🛠 Tech Stack
- Python
- OpenCV
- PyTorch
- Streamlit

## ⚡ How to Run
pip install -r requirements.txt
streamlit run app.py

## 🧠 Inspiration
Understanding animal behavior is critical in ecology and conservation.
This tool bridges AI with biological research.

## 🔮 Future Improvements
